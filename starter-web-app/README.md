# Starter Web App

This folder contains a small browser app you can use as the first version of a new project.

## What it does

- Adds, completes, and deletes tasks
- Stores tasks in browser local storage
- Runs without installing dependencies

## Run it

Open `index.html` in your browser.

If you want a simple local server instead, run this from the `starter-web-app` folder:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Files

- `index.html` for the page structure
- `styles.css` for the UI
- `app.js` for the app logic
